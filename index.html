Countdown

This is a simple drop-in progressive web component which does a countdown.

Burning Man 2019 Opens on Aug 25;

Instructions

Simply include the javascript file; 
<script src="https://cjtrowbridge.com/projects/Countdown/countdown.js"></script>
And then include an element with the data-countdown attribute and the countdown class. 
<div class="countdown" countdown="D M d Y H:i:s O"></div>
For a countdown, include a progress element with the countdown class, as well as data-from and data-to. 
<progress class="countdown" data-from="D M d Y H:i:s O" data-to="D M d Y H:i:s O"></div>
